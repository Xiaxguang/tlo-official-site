潮流抽卡冒險 官方網站入口頁

檔案說明：
- index.html：正式官網首頁，含「開始遊戲」入口按鈕
- game.html：遊戲入口佔位頁，正式上線時可用你的真正遊戲檔案取代
- about.html：遊戲介紹
- privacy.html：隱私權政策
- terms.html：使用條款
- contact.html：聯絡方式
- rewarded-ads.html：廣告獎勵說明
- assets/styles.css：網站樣式
- assets/site.js：手機選單功能

上線前你需要修改：
1. 將所有「請填入你的聯絡 Email」改成你的正式客服信箱。
2. 將你的真正遊戲主檔改名為 game.html，取代目前的 game.html。
   或者打開 index.html，把 href="game.html" 改成你的遊戲網址。
3. 確認每個頁面都能正常打開。
4. 確認手機版排版正常。
5. 隱私權政策頁建議不要放廣告碼，保持可公開瀏覽與乾淨內容。
